# Generated Outputs

This directory stores artifacts produced by the experiment scripts.

## `predictions/`

Generated by:

```bash
python experiments/run_baseline.py --method no_rag
python experiments/run_baseline.py --method vector_rag
python experiments/run_baseline.py --method toc_rag
python experiments/run_baseline.py --method hybrid_rag
python experiments/run_ablation.py
```

Main files:

- `*_predictions.csv`: per-question predictions and metrics for each method.
- `*_summary.json`: aggregate summaries for each method.
- `ablation/*.json`: small-grid ablation summaries.

These files can be regenerated and may be ignored by Git depending on the submission policy.

## `analysis/`

Generated by:

```bash
python experiments/analyze_outputs.py
```

Main files:

- `aligned_predictions.csv`
- `overall_metrics.csv`
- `paired_differences.csv`
- `paired_bootstrap_tests.json`
- `analysis_report.md`
- `fig_cost_vs_quality_f1.png`
- `fig_cost_vs_citation.png`

Keep this directory when submitting a reproducibility snapshot, or regenerate it from the four baseline prediction CSVs.
